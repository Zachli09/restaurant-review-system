# Restaurant Review System

A console-based C++ application that allows users to log in and review restaurants through a menu-driven interface.

## Features

* User login authentication using usernames and passwords
* Store restaurant reviews in text files
* Prevent duplicate reviews from the same user
* Display all reviews from all users
* Calculate average ratings for restaurants
* Menu-driven user interface

## Technologies Used

* C++
* File Handling
* Functions
* Loops and Conditionals
* Input Validation

## What I Learned

This project helped me practice working with file input/output, user authentication systems, data management, and modular programming in C++.
